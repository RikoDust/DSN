# DAY'S NOTES - PRIVE


## 📋 À propos

Projet de création d'une application de notes

Contexte : Créer une application de notes differente de la plupart de ce type d'application

## ✨ Fonctionnalités principales

- Saisir des notes facilement
- Créer des note par type : texte, liste, contact, lieux, rdv
- notes intéractives (appel, msg, gps, agenda) depuis celles-ci
- Toutes les notes sont partageable par msg, mail ou autres
- Darkmode disponible
